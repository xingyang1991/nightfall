# 示例与模板

## 触发语句样例
- 想逛逛小众街区，买个小物。
- 我想找点对味的小玩意，别太贵。
- 给我一个不挤的内街，随便看看也行。

## 首轮候选卡模板
```
[候选 X] 片区名：一句话氛围
你去主要看：品类/小物（含大致价位）
预计耗时：XX–XX 分钟 | 拥挤风险：低/中/高
你会喜欢的细节：1–2 条
```

可选轻按钮：更安静 / 更便宜 / 更近 / 随缘换一组

## 最终方案模板
```
你选了：片区名（时间）
【动线】2–3 站就够：
1) 站名（看什么）
   - 选购小提示：…
   - 不尴尬退场：…
2) 站名（看什么）
   - 选购小提示：…
   - 不尴尬退场：…
【买什么更不后悔】按材质/用途/体积选一个
【Plan B】下雨/太挤/不想买时的室内替代或“只摸材质任务”
【今日票根】主题：…（邀请补一句触感或印象）
```

## 示例（简版）
[候选 1] 小巷陶器：釉面干净，手感克制
你去主要看：小盘/杯子（200 内）
预计耗时：45–60 分钟 | 拥挤风险：低

[候选 2] 材质杂货：陶 + 木 + 亚麻，陈列很克制
你去主要看：托盘/香插/小碟
预计耗时：30–45 分钟 | 拥挤风险：中

你选了：小巷陶器（45–60 分钟）
【动线】2 站就够：
1) 小花器店（先看杯沿与底部收口）
   - 选购小提示：手感顺 + 口沿薄更稳
   - 不尴尬退场：我再逛一圈回来
2) 小盘角落（挑每天用的：放钥匙或茶包）
   - 选购小提示：好擦、可进洗碗机
【Plan B】雨天就去附近书店坐 30 分钟再决定
【今日票根】主题：雾面白釉（补一句今天摸到的质感）
