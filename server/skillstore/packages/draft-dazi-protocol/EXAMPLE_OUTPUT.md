# 搭子不尴尬协议 · 生成样例（v0.1）

> 这是一个“从 Stage A 候选 → 用户选择 → Stage B 定稿”的完整样例。  
> 地点/机构名称为占位符，目的是展示**表达方式与流程**。

## Stage A 候选卡（CandidateDeck）

我先给你一组候选，你点一个，我再把它写成“能直接出发”的定稿页。  
（你也可以点开关：更近 / 更安静 / 更稳）

**沉默散步30分钟**
- 适合你：想一起出门，但不想一直说话
- 风险/别踩：别踩：提前说好‘只在最后交换三句’
- 选择：生成这份协议

**并排共工60分钟**
- 适合你：各做各的，但有共处感
- 风险/别踩：别踩：别坐面对面，容易变聊天局
- 选择：生成这份协议

**一小时同频活动**
- 适合你：看展/翻杂志/逛一排小店
- 风险/别踩：别踩：别排超过10分钟队，超时就切Plan B
- 选择：生成这份协议

> 用户选择：第 2 张（或你最顺眼的一张）

### Stage B 定稿（系统输出）

**结局卡**
- 主动作：把‘并排共工60分钟’协议发给搭子
- Plan B：如果对方不方便，就改成单人 60 分钟点灯专注
- 为什么对你成立：你想要共处感，但不想经营关系；协议先把边界搭好。
- 风险旗帜：可能人多 / 营业时间有变（信息不足时自动降级）
- 最后 300 米：开场句：‘我们就各做各的，结束再交换三句就好。’
- 怎么待最舒服：退出句：‘我先撤了，今天到这儿就很刚好。’
- 切 Plan B 条件：聊到第三句就收束。

**PocketTicket（默认私密）**
- 标题：今晚相处得很轻
- 时间：2026-02-04 21:20
- 我选了什么：把‘并排共工60分钟’协议发给搭子
- 印章：📎
- 轻反馈：喜欢 / 一般 / 不喜欢（原因：太吵 / 没座 / 太远 / 太贵）

**FootprintEntry（默认仅自己可见）**
- 标题：今晚相处得很轻
- 片段：做完了就收束，没有拉长
- 情绪：修复 / 轻探索 / 低压共处（任选）
