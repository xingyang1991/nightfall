# Content Spec

## Risk Flags 词典（<=10）

- `✓ 比较稳`：营业/座位信号稳定
- `✓ 可久坐`：评论或规则明确可停留
- `✓ 安静倾向`：近期评论噪音负面少
- `✓ 插座可能`：评论或图片暗示有电
- `⚠️ 可能限时`：限时关键词出现
- `⚠️ 排队波动`：近期出现排队或等位
- `⚠️ 噪音偏大`：评论出现吵/拥挤
- `⚠️ 不太电脑友好`：禁电脑/禁久坐信号
- `⚠️ 营业不稳`：营业时间来源不一致
- `⚠️ 低消偏高`：最低消费/人均偏高

## Checklist 句式库（<=10）

- `坐哪：靠墙或里侧位更稳，避开入口风口。`
- `坐哪：吧台末端更安心，视线不被打断。`
- `坐哪：窗边更亮，适合短暂停留。`
- `点啥：点一杯基础款，待久也不尴尬。`
- `点啥：轻食或小点更稳，桌面不乱。`
- `待多久：建议 45–90 分钟，不易被催。`
- `待多久：适合 60–120 分钟，超过可能被提醒。`
- `进门：先看是否有空位，再问是否限时。`
- `进门：柜台先问“能坐多久/能否用电脑”。`
- `满座：直接切换 Plan B，不要硬等。`
