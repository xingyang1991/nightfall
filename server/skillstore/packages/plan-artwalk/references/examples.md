# Examples

## Sample Prompts

- "Give me a 90-minute Artwalk."
- "两个人出去走走但不想硬聊。"
- "书店器物风，90 分钟。"

## First-Round Route Cards

```
[Route 1] Object Shop Light → Magazine Aisle → Sit For A Drink (80-100 min)
Opening: A small object shop with good natural light
Ending: A quiet cafe with window seats
Risk: Hours may vary; confirm before going

[Route 2] Tiny Exhibit (20 min) → Corner Facade → Bookstore Nook (90-110 min)
Opening: Small gallery or pop-up exhibition
Ending: Bookstore corner seat
Risk: Exhibit may be seasonal; have a bookstore fallback
```

## Final Plan (After User Pick)

```
Opening — Object Shop Light
- Why stop: Materials and light slow the pace immediately
- Optional task: Touch one material and describe the texture

Middle — Magazine Aisle
- Why stop: Quiet browsing keeps conversation low-stakes
- Optional task: Pull one line from a page

Ending — Sit For A Drink
- Why stop: A seated ending seals the walk
- Optional task: Name the walk like a film title

Camera tasks (optional):
1) Shoot a reflection or shadow
2) Frame a doorway or threshold

Mini-map:
A -> B (8 min) -> C (6 min)

Plan B (rain/crowds):
Bookstore table -> Stationery aisle -> Quiet cafe corner

Echo:
“Aisle Of Soft Light”
```
