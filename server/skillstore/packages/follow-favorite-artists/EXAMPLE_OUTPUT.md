# 盯住我爱的艺术家（艺术家雷达） · 生成样例（v0.1）

> 这是一个“从 Stage A 候选 → 用户选择 → Stage B 定稿”的完整样例。  
> 地点/机构名称为占位符，目的是展示**表达方式与流程**。

## Stage A 候选卡（CandidateDeck）

我先给你一组候选，你点一个，我再把它写成“能直接出发”的定稿页。  
（你也可以点开关：更近 / 更安静 / 更稳）

**艺术家 A**
- 你可能会喜欢：你喜欢其冷静的材质与留白
- 风险/别踩：风险：同名较多，需二次确认
- 选择：盯这个人（进入定稿）

**艺术家 B**
- 你可能会喜欢：线条感强，适合‘只看一个点’
- 风险/别踩：风险：展讯更新慢，可能需替代
- 选择：盯这个人（进入定稿）

**艺术家 C**
- 你可能会喜欢：更轻更日常，适合下班看一眼
- 风险/别踩：风险：可能只在线上出现
- 选择：盯这个人（进入定稿）

> 用户选择：第 2 张（或你最顺眼的一张）

### Stage B 定稿（系统输出）

**结局卡**
- 主动作：选中一个候选并出发
- Plan B：不成立就切到兜底
- 为什么对你成立：更稳、更省事。
- 风险旗帜：可能人多 / 营业时间有变（信息不足时自动降级）
- 最后 300 米：最后300米：看提示走。
- 怎么待最舒服：怎么待：坐下就别换。
- 切 Plan B 条件：不舒服就切 Plan B。

**PocketTicket（默认私密）**
- 标题：今晚做了一件小事
- 时间：2026-02-04 21:20
- 我选了什么：选中一个候选并出发
- 印章：🧑‍🎨
- 轻反馈：喜欢 / 一般 / 不喜欢（原因：太吵 / 没座 / 太远 / 太贵）

**FootprintEntry（默认仅自己可见）**
- 标题：今晚做了一件小事
- 片段：做完了就收束，没有拉长
- 情绪：修复 / 轻探索 / 低压共处（任选）
