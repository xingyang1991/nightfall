# 搜索方向与调用工具

## 搜索方向（要搜什么）

- 识别雨天状态（天气接口或用户手动开启雨天模式）。
- 优先检索室内锚点：连廊/地铁上盖/大堂/图书馆/商场中庭/室内展览。
- 提取入口动线线索：地下入口、雨棚、连廊、临停、地库直达等。
- 雨天拥挤惩罚：对雨天峰值易爆点降权，并强配 Plan B。
- 估算“淋雨成本”（粗分钟数）用于候选排序。

## 调用工具（用什么搜/怎么验证）

- 天气状态获取器或雨天模式开关。
- POI 搜索器（室内优先）+ 到达时间估算器。
- 入口/动线线索提取器（文本 + 图片）。
- 雨天拥挤信号提取器（评论关键词 + 时段）。
- 营业时间多源校验器（雨天更频繁）。
