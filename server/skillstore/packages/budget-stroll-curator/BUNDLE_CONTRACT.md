# Curatorial Bundle 最小输出合同 v0.1

> 本文件是该 Skill 的**输出接口**。实现必须严格遵守 **Stage A / Stage B** 的产物要求。  
> 设计依据：Coffice 的“结局函数（Outcome Function）而非信息流”、多频道策展面板消费、低打扰/强退路/强隐私。  

- 更新日期：2026-02-04
- 默认原则：宁可更稳、更少，也不冒险、更不刷。


## 本 Skill：千元以内逛逛（预算买手）

### 本技能的 P0 输出要点

- Stage A 候选规模：Stage A 建议先给 6 张“补丁类型卡”（只买一件/只看不买/纸品/器物/中古/香味灯），用户选类型后再给 4–6 张候选店。
- Stage A 交互开关：推荐开关：只看不买 / 只买一件 / 更安静 / 更近。
- Stage B 收束重点：Stage B 必须写“超预算撤退规则”一句，并提供“零消费也成立”的 Plan B。
- 派生素材建议：建议 ImagePack（纸袋/材质封面）；MapPack 可选。
- PocketTicket 默认印章：🧷

## Stage A：CandidateDeck（让用户选）

Stage A 的目标：给用户一组**能选**的候选卡，让用户用很小的参与成本把方向“拧准”。  
它不是信息流，不允许无限刷新刷到爽；默认克制、可展开但要节流。

### CandidateDeck 必须包含
- 4–6 张候选卡（除非本 Skill 特殊说明）
- 每张候选卡都要写清：
  - **为什么对我成立**（1 句）
  - **可能哪里翻车/哪里别踩**（1 句）
  - **一个动作**：选它进入 Stage B（必要时附：地图预览/展开更多）

### 允许的轻追问（最多 2 个）
只问能显著改变候选结果的问题：
- 交通方式：步行 / 地铁 / 开车
- 停留时长：30–60 / 60–120 / 120+
- 社交电量：完全隐身 / 轻有人气 / 不介意热闹
- 预算段：不设 / 低 / 中

### 节流规则（避免信息流化）
- 默认只展示 4–6 张
- “展开更多”最多 1–2 次；每次展开都要提示：更多意味着核验更弱、翻车概率更高
- 若候选不足：必须给“救援卡”（扩大范围 / 换业态 / 最稳兜底）


### 本技能的 Stage A 细则

- 候选规模与分组：Stage A 建议先给 6 张“补丁类型卡”（只买一件/只看不买/纸品/器物/中古/香味灯），用户选类型后再给 4–6 张候选店。
- 顶部开关建议：推荐开关：只看不买 / 只买一件 / 更安静 / 更近。

## Stage B：OutcomeCard + Plan B + PocketTicket + FootprintEntry（收束成结局）

Stage B 的目标：把体验收束成“一张可执行结局卡”，并且默认带退路与沉淀闭环。

### 必须产出
1) **OutcomeCard（结局卡）**
- 主动作：一键开始（导航/打开播放/开始计时/复制协议等）
- Plan B：并列退路，一键切换（不是失败）
- 理由：最多 2 句，只写关键事实，不写广告腔
- 风险旗帜：0–2 条，宁可保守
- 外出类额外 3 句：
  - 最后 300 米怎么到
  - 怎么待最舒服
  - 触发切 Plan B 的条件（例如：等位 > 10 分钟就切）

2) **PocketTicket（口袋票根，默认私密）**
- 标题（短）
- 时间
- 我选了什么（主动作）
- 印章（stamp）
- 轻反馈：喜欢/一般/不喜欢 + 原因标签（最多 1 个）

3) **FootprintEntry（足印回声，不排名不比较）**
- 一句事实标题（不鸡汤）
- 1–2 句发生了什么（可空）
- 情绪标签（可选）
- 默认隐私等级（默认仅自己可见）

### 可选派生（按本 Skill 的频道需求）
- MapPack：主动作与 Plan B 的地图深链 + 到达提示
- ImagePack：合规图片或“材质/色块封面”占位（必须过风控，去 EXIF）
- AudioLine：10–20 秒旁白（给车机/电台的一瞥版）


### 本技能的 Stage B 细则

- 必写项：Stage B 必须写“超预算撤退规则”一句，并提供“零消费也成立”的 Plan B。
- PocketTicket stamp 默认：🧷

## 多频道切片建议（同一份 Bundle，多种镜头语言）

> 前端频道只消费素材，不承载 Skill 逻辑。

- 发现：OutcomeCard（中密度，1 主 + 1 退）
- 酒保：Stage A 的点单式追问 + Stage B 的收束卡
- 电台：AudioLine + 主动作按钮（低密度）
- 星图：仅聚合符号/波纹（不指向人）
- 屏保：ImagePack（近零密度，每日最多更新一次）
- 楼下：一句短句或一张短图（强节流）
- 足印：FootprintEntry + 周报汇总入口
## Slice Seeds（推荐，用于自动多频道切片）

> 说明：Stage A / Stage B 的必产物是硬门槛；下面这些“种子”是为了让电台/屏保/楼下/星图切片更稳定。  
> 缺失不报错，但会导致切片更模板、更像通用文案。

- mood_tag：轻探索
- tone_keywords：小补丁/不冲动/手感
- symbol_hint：🧷
- visual_motif：小物静物+柔和阴影
- audio_motif：纸袋摩擦声+一句提示
- downstairs_line_seed（≤18字）：只买一件就够
