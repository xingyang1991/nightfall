# 书里偷闲 · 生成样例（v0.1）

> 这是一个“从 Stage A 候选 → 用户选择 → Stage B 定稿”的完整样例。  
> 地点/机构名称为占位符，目的是展示**表达方式与流程**。

## Stage A 候选卡（CandidateDeck）

我先给你一组候选，你点一个，我再把它写成“能直接出发”的定稿页。  
（你也可以点开关：更近 / 更安静 / 更稳）

**安静型书店/空间 A**
- 适合你：能坐住，不需要解释
- 风险/别踩：风险：晚间可能有活动
- 选择：选它定稿

**灯光舒服的书店 B**
- 适合你：不刺眼，适合读一段
- 风险/别踩：风险：插座不一定多
- 选择：选它定稿

**座位最稳的空间 C**
- 适合你：更像公共客厅
- 风险/别踩：风险：人来人往
- 选择：选它定稿

**兜底：酒店大堂 D**
- 适合你：最不容易翻车
- 风险/别踩：风险：不那么‘像书店’
- 选择：选它定稿

> 用户选择：第 2 张（或你最顺眼的一张）

### Stage B 定稿（系统输出）

**结局卡**
- 主动作：去书店 B 坐 60–90 分钟，只读一段
- Plan B：如果开始活动变吵，就切到酒店大堂 D
- 为什么对你成立：你不需要消费也能待住；灯光和人声更稳。
- 风险旗帜：可能人多 / 营业时间有变（信息不足时自动降级）
- 最后 300 米：进门直奔杂志区边角位，最不尴尬。
- 怎么待最舒服：只做一件事：读 10 页或写 5 行。
- 切 Plan B 条件：周围持续聊天声 > 3 分钟就切 Plan B。

**PocketTicket（默认私密）**
- 标题：今晚借书店透口气
- 时间：2026-02-04 21:20
- 我选了什么：去书店 B 坐 60–90 分钟，只读一段
- 印章：📚
- 轻反馈：喜欢 / 一般 / 不喜欢（原因：太吵 / 没座 / 太远 / 太贵）

**FootprintEntry（默认仅自己可见）**
- 标题：今晚借书店透口气
- 片段：做完了就收束，没有拉长
- 情绪：修复 / 轻探索 / 低压共处（任选）

## 派生切片（示例，给前端各频道消费）

**DownstairsSnippet（楼下一句）**  
去书里躲一下

**RadioLine（电台旁白，10–20秒）**  
今晚先去书里躲一下。主动作在卡片里，随时可切 Plan B。

**WallpaperCover（屏保提示）**  
- cover_type：typography_card（兜底）  
- visual_motif：纸张纹理+留白排版

**StarMapSymbolHint（星图符号提示，最终需聚合呈现）**  
- symbol_hint：📚  
- mood_tag：安静
