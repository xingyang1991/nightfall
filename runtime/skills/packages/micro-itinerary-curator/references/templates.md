# 模板与样例（走一圈也算旅行）

## 标题与句式
- 标题模板（单行）
- {30/60/90/120}分钟行程：{主题}
- 走一圈也算旅行：两站就够
- 别做攻略：做时间块

- 理由句模板（≤2句）
- 只做2-3站，走得通，收得回。
- 每站只做一件事，不把行程做成任务。
- 崩了也不尴尬：排队/天气/体力都有退路。

- Need-to-know 句式（≤5条）
- 第一站只做一个动作，别加戏。
- 排队超过10-15分钟，跳过本站。
- 每站留10分钟缓冲。
- 最后一站必须是收束点。
- 只做第一站也算完成。

- 风险短写（候选页每卡≤1条）
- ⚠️ 时间窗偏窄
- ⚠️ 排队连锁风险
- ⚠️ 需预约/门票
- ⚠️ 天气敏感
- ✓ 比较稳

## 候选页结构
- 标题：走一圈也算旅行
- 副标题：你附近 · 想出去一下 · 90分钟以内
- chips：90分钟｜两站就够｜可收束
- 布局建议：首屏12张（3组×4张），支持「换一批」与「按更近/更安静/更稳/更便宜」重排
- 呈现：卡片墙 + 底部迷你地图条（方向感即可，点 pin 高亮卡片；候选页不展开导航细节）

- 分组标题示例
- 最稳的一组（两站就够）
- 更有主题的一组
- 更省事的一组（只做第一站也成立）

- 卡片格式（单卡）
- 【{示例名}（{时长}min）】
- 步行/地铁/开车≤{分钟} · {站数}站
- 标签：行程 / {主题} / {质地}
- 怎么用：{单一动作或跳过条件}
- {风险短写}

## 定稿页结构
- 你点了：{选中卡片名}
- 按钮：开始这条行程 / 切Plan B / 换一条行程
- 两句理由（事实型）
- 怎么待最舒服（3条动作）
- Need-to-know（≤5）
- 风险旗帜（≤2）
- Plan B（并列退路）
- 呈现方式：
- 地图：路线小图 + 退出点
- 文字：时间线（00:00-00:30 / 00:30-01:30）
- 图片：两站证据图

## 图片与地图规则
- 候选页像路线封面：编号1-3 + 主题小图标
- 定稿页给小地图：线路 + 站点编号 + 退出点
- 每站配证据图（门口/展签/桌角）
