### Stage B 定稿（系统输出）

**结局卡**
- 主动作：去「久坐频段」的店 A，点一杯不太打扰睡眠的
- Plan B：如果店 A 太忙，就切到店 B（同频段）
- 为什么对你成立：你要的是‘续航’，不是折腾；这两家都能坐住。
- 风险旗帜：可能人多 / 营业时间有变（信息不足时自动降级）
- 最后 300 米：点单一句：‘一杯低因/茶咖，冰/热都行，我想坐一会儿。’
- 怎么待最舒服：坐法：先找能看到门口但不在动线的位置。
- 切 Plan B 条件：排队 > 10 分钟就切 Plan B。

**PocketTicket（默认私密）**
- 标题：今晚把自己调到了合适频段
- 时间：2026-02-04 21:20
- 我选了什么：去「久坐频段」的店 A，点一杯不太打扰睡眠的
- 印章：☕️
- 轻反馈：喜欢 / 一般 / 不喜欢（原因：太吵 / 没座 / 太远 / 太贵）

**FootprintEntry（默认仅自己可见）**
- 标题：今晚把自己调到了合适频段
- 片段：做完了就收束，没有拉长
- 情绪：修复 / 轻探索 / 低压共处（任选）

## 派生切片（示例，给前端各频道消费）

今晚先把自己调个频。主动作在卡片里，随时可切 Plan B。

**StarMapSymbolHint（星图符号提示，最终需聚合呈现）**  
- symbol_hint：☕️  
- mood_tag：补给
