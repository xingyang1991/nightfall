# 展后留评 · 生成样例（v0.1）

> 这是一个“从 Stage A 候选 → 用户选择 → Stage B 定稿”的完整样例。  
> 地点/机构名称为占位符，目的是展示**表达方式与流程**。

## Stage A 候选卡（CandidateDeck）

我先给你一组候选，你点一个，我再把它写成“能直接出发”的定稿页。  
（你也可以点开关：更近 / 更安静 / 更稳）

**只摘一句**
- 适合你：你想留痕，但不想写太多
- 风险/别踩：别踩：不要解释作品，只摘你真的记住的那句
- 选择：用这个方式生成票根

**三事实票根**
- 适合你：不写感想，只写发生了什么
- 风险/别踩：别踩：每条≤12字，越短越稳
- 选择：用这个方式生成票根

**一张拼贴**
- 适合你：把两张图+一句话拼成一页
- 风险/别踩：别踩：没有合规图就用色块/纹理占位
- 选择：用这个方式生成票根

**30秒语音**
- 适合你：说完就结束，不加工
- 风险/别踩：别踩：别复盘太长，30秒到点收束
- 选择：用这个方式生成票根

> 用户选择：第 2 张（或你最顺眼的一张）

### Stage B 定稿（系统输出）

**结局卡**
- 主动作：用‘三事实票根’把展折成一页
- Plan B：如果写不出来，就只摘一句最像你的展签
- 为什么对你成立：你想留下些什么但不想经营；事实比感想更稳。
- 风险旗帜：可能人多 / 营业时间有变（信息不足时自动降级）
- 最后 300 米：事实 1：我看了什么（≤12字）
- 怎么待最舒服：事实 2：我停在哪个点（≤12字）
- 切 Plan B 条件：事实 3：我带走了什么（≤12字）

**PocketTicket（默认私密）**
- 标题：今晚把展折成了一页
- 时间：2026-02-04 21:20
- 我选了什么：用‘三事实票根’把展折成一页
- 印章：🧾
- 轻反馈：喜欢 / 一般 / 不喜欢（原因：太吵 / 没座 / 太远 / 太贵）

**FootprintEntry（默认仅自己可见）**
- 标题：今晚把展折成了一页
- 片段：做完了就收束，没有拉长
- 情绪：修复 / 轻探索 / 低压共处（任选）
