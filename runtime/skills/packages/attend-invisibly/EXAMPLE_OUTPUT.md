# 不聊天也参与（隐形参与） · 生成样例（v0.1）

> 这是一个“从 Stage A 候选 → 用户选择 → Stage B 定稿”的完整样例。  
> 地点/机构名称为占位符，目的是展示**表达方式与流程**。

## Stage A 候选卡（CandidateDeck）

我先给你一组候选，你点一个，我再把它写成“能直接出发”的定稿页。  
（你也可以点开关：更近 / 更安静 / 更稳）

**完全旁观模式**
- 适合你：你只在场，不进入任何对话链
- 风险/别踩：可能的别踩：别站在入口动线，容易被搭话
- 选择：用这个模式定稿

**极轻互动模式**
- 适合你：你可以点头打个招呼，但不需要自我介绍
- 风险/别踩：可能的别踩：聊到第三句就收束，避免被拉长
- 选择：用这个模式定稿

> 用户选择：第 2 张（或你最顺眼的一张）

### Stage B 定稿（系统输出）

**结局卡**
- 主动作：用‘完全旁观模式’去参加活动，停 25 分钟就走
- Plan B：如果被搭话或场面太吵，走到门口外侧直接撤
- 为什么对你成立：你想在场，但不想进入对话链；旁观也算参与。
- 风险旗帜：可能人多 / 营业时间有变（信息不足时自动降级）
- 最后 300 米：站位：别站入口，选侧边靠墙。
- 怎么待最舒服：手里拿一杯水/册子，减少被搭话概率。
- 切 Plan B 条件：10 分钟内出现两次搭话，就切 Plan B。

**PocketTicket（默认私密）**
- 标题：今晚在场但没被耗尽
- 时间：2026-02-04 21:20
- 我选了什么：用‘完全旁观模式’去参加活动，停 25 分钟就走
- 印章：🫥
- 轻反馈：喜欢 / 一般 / 不喜欢（原因：太吵 / 没座 / 太远 / 太贵）

**FootprintEntry（默认仅自己可见）**
- 标题：今晚在场但没被耗尽
- 片段：做完了就收束，没有拉长
- 情绪：修复 / 轻探索 / 低压共处（任选）
